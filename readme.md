### Шаг 1
Склонировать репозиторий себе на ПК

`git clone <link>`

### Шаг 2
Создать папку в репозитории

`ПКМ > "Создать папку"` или в консоли `mkdir new_folder`

### Шаг 3
Посмотреть статус репозитория, добавить новую папку в индекс

`git add .` > `git commit -am 'message'`

### Шаг 4
Запушить изменения

`git push`
